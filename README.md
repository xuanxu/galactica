# Galactica

Galactica is a galactic chemical evolution model. Its code solves a system of first-order integro-differential equations for the chemical evolution of a spiral galaxy, assuming an analytical star formation (SF) law, an initial mass function (IMF), stellar lifetimes and nucleosynthetic yields.
